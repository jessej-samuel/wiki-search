## Features 🔥
- [x] Dark mode 😎
- [x] Lookup cat articles 😺
- [x] Search random articles 🎲
- [x] Focus with `/` 🧑‍💻
- [x] Confetti 🎊

## Run it locally 💻
- Clone the repository into a folder
- Install dependencies using:
```bash
cd wiki-search
npm install
npm start
```
- Voila! The app should be running on http://localhost:3000/ :)
